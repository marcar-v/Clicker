# README — Clicker 2D de Formas Geométricas

**Nombre:** Clicker 2D de Formas Geométricas  
**Tipo de juego:** Arcade / Casual / 2D  
**Tecnología:** Unity 2D  
**Plataforma:** PC  

## Descripción
Juego de clicker en 2D donde el jugador interactúa con formas geométricas. El objetivo es alcanzar la mayor distancia posible mientras las formas se mueven y reaccionan a los clicks. Incluye animaciones y música de fondo en la versión final.  

## Controles
- **Clic izquierdo:** Interactuar con las formas.  
- **Menú principal:** Selección de juego y visualización de mejor distancia.  

## Características
- Animaciones de las formas geométricas.  
- Registro de la mejor distancia alcanzada.  
- Menú principal funcional y flujo entre juego y menú.  
- Música de fondo integrada (versión final).
